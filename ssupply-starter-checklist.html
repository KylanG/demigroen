SSUPPLY Starter v1.0 — Klantproject checklist

Doorloop alle stappen om een nieuw klantproject op te zetten.

0 van 22 voltooid
Repository aanmaken
Stap 1
Ga naar github.com/KylanG/ssupply-starter en klik op "Use this template" → "Create a new repository"
Stap 2
Vul de repository naam in (bijv. klantnaam-website) en zet op Private
Stap 3
Klik op "Create repository"
Lokale setup
Stap 4
Kloon de repository lokaal: git clone https://github.com/KylanG/[naam].git
Stap 5
Ga de map in: cd [naam]
Stap 6
Installeer dependencies: npm install
Stap 7
Maak .env.local aan: cp .env.local.example .env.local
Omgevingsvariabelen — Sanity
Stap 8
Vul NEXT_PUBLIC_SANITY_PROJECT_ID in met het Sanity project ID van de klant
Stap 9
Vul NEXT_PUBLIC_SANITY_DATASET in (standaard: production)
Stap 10
Maak een Viewer API token aan in Sanity en vul SANITY_API_READ_TOKEN in
Stap 11
Maak een Editor API token aan in Sanity en vul SANITY_API_WRITE_TOKEN in
Stap 12
Voeg http://localhost:3000 toe aan Sanity CORS Origins (met credentials)
Omgevingsvariabelen — E-mail
Stap 13
Vul RESEND_API_KEY in met de Resend API key
Stap 14
Vul CONTACT_FORM_TO_EMAIL in met het e-mailadres van de klant
Project starten
Stap 15
Voer het seed script uit: npm run seed
Stap 16
Start de dev server: npm run dev
Studio configuratie
Stap 17
Open http://localhost:3000/studio en vul Algemene Instellingen in
Stap 18
Vul Header Instellingen in (logo, navigatie, CTA knop)
Stap 19
Vul Footer Instellingen in (kolommen, socials, copyright)
Stap 20
Maak de homepage aan in Pagina's met slug home
Aanpassingen & live zetten
Stap 21
Pas --color-primary aan in app/globals.css naar de merkkleur van de klant
Stap 22
Zet siteNoIndex uit in Algemene Instellingen → SEO zodra de site live gaat
